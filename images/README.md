# Project photos

The homepage "Recent Work" gallery (`index.html`) expects these 5 image files.
Drop the photos in here using these exact filenames and the gallery renders automatically.

| Filename | Photo |
|---|---|
| `cedar-board-on-board-privacy-fence.jpg` | Stained cedar board-on-board privacy fence with cap & trim (vertical boards) |
| `horizontal-cedar-fence-gate.jpg` | Horizontal cedar privacy fence with matching gate along a driveway |
| `wrought-iron-ranch-fence.jpg` | Black ornamental iron ranch fence bordering an open field |
| `cedar-privacy-fence-steel-posts.jpg` | Dog-ear cedar privacy fence built on galvanized steel posts |
| `horizontal-cedar-fence-sloped-yard.jpg` | Horizontal cedar fence with gate stepped down a sloped front yard |

Tips:
- JPG, roughly 1600 px on the long edge keeps the page fast.
- The gallery crops to a 4:3 tile, so the subject can sit anywhere in frame.
- To add more, copy a `<figure class="shot">` block in `index.html` and add the file here.
